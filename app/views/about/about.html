Adobe Publish for Wordpress

Folio Author for WordPress is a plugin for WordPress that enables publishers to create HTML articles and folios for Adobe Digital Publishing Suite (DPS), directly from their WordPress instance. Using this plugin, publishers can streamline the creation and management of their content on DPS apps across multiple devices.

Known Issues

If you receive an error that the version is different or your OATH (token) has expired, please try the call again and it should work. We're capture and try to resolve these conflicts. In the future the plugin will try the request again with the working version / oauth token. But this beta build you will need to make the call again.
